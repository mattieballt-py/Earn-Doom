### What to build
Count Instagram content by observing the visible feed layout, not just clicks or route changes, so the counter advances as the user scrolls through posts, reels, and stories in the main content area.

### Acceptance criteria
- The counter increases when the visible Instagram content item changes in the main feed layout.
- The counter remains stable for the same visible item across repeated scans.
- The counter continues to persist across sessions using the existing storage model.
